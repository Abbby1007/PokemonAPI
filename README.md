# PokemonAPI
In this project I used PokiApi to make a name and type Pokemon Quiz.
Just click the link and choose a quiz.

The demo link: https://abbby1007.github.io/PokemonAPI/

I made the game with the purpose of learning about APIs. When I watch on tutorial video I found myself needing to watch four other videos to understand different concepts. It was really fun to work with outside data and I look forward to watching other API videos.
